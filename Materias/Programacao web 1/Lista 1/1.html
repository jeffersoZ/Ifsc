Internet vs Web

A Internet é uma rede global que conecta computadores, enquanto a Web é um serviço que a Internet oferece. A Web é um sistema de informações interligadas, que permite o acesso a conteúdos através da Internet.

História da Internet e Web

A Internet surgiu nos anos 1960 com a ARPANET. A Web foi criada por Tim Berners-Lee em 1989.

  • 1969 - ARPANET
  • 1983 - TCP/IP
  • 1989 - Web criada
